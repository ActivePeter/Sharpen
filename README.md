Sharpen is a low-performance C++ network library

Requires(Windows):
  Windows version >= 7.
  Msvc >= 19.0 or Clang >= 3.5

Requires(Linux):
  Linux kernel version >= 2.6.28.
  GCC >= 4.8 or Clang >= 3.5

To build, run:
  ./build.sh
  ./build.cmd
